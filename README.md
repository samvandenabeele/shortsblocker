# shortsblocker
a chrome extension that blocks youtube shorts, instagram reels, and hides tiktok

Since I'm really addicted to YouTube Shorts, Instagram Reels, and TikTok, and I want to regain the endless amounts of time I spend on there instead of studying and coding, I quickly created this chrome extension to block them on my laptop.

I also added a feature to enable/disable it, and will add scheduling etc too, and maybe a way to prevent you from disabling it whenever you want, and instead having to beg your friends
